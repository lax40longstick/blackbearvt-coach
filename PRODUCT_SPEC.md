# Bear Den Coach HQ Product Spec

## Product summary

Bear Den Coach HQ is coaching software for youth hockey teams. It combines practice planning, drill management, roster operations, and bench-ready execution in one workspace.

## Primary buyer

Default v1 target:

- Team staff

Secondary targets later:

- Solo head coaches
- Club directors

## Core promise

Help coaches get from planning to practice execution faster, with less scattered information and less manual admin.

## v1 outcomes

- A coach can create an account without manual configuration
- A coach can create or join a team workspace
- A coach can manage roster, drills, and practice plans in one place
- A coach can run an on-ice practice flow from a phone
- A paying customer can manage billing and account settings

## Core v1 features

- Email/password authentication
- Team and organization ownership
- Practice generator
- Drill library and diagram support
- Roster and line management
- Account and billing surfaces
- Basic onboarding
- Export and backup path

## v1 non-goals

- Full club-wide analytics
- Complex director reporting
- Native mobile apps
- Deep parent/player communication suite
- Video analysis

## User roles

- Owner
- Head coach
- Assistant coach
- Manager
- Director

## Commercial packaging

- Starter: solo coach, one team
- Team: shared staff workspace
- Club: multi-team organization

## Success metrics

- Signup completion rate
- Onboarding completion rate
- First practice saved
- Weekly active teams
- Trial-to-paid conversion

## Risks

- Auth and team setup friction
- Billing before clear value is proven
- Trust concerns around data ownership and recovery
- Prototype-era app structure slowing iteration
