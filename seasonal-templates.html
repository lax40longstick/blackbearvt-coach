The phase model

A youth season has 4 phases, not 30 weeks of "practice"

Treating every week the same is the most common planning mistake youth coaches make. Players need different things in October than they need in February. Here's the model.

Phase 1 — Preseason (Aug-Sept)

Skating, edges, conditioning, individual skills. Almost no team systems. Build the engine before installing the structure. Daily skating focus, station-heavy practices, low compete pressure.

Phase 2 — Install (Oct-Nov)

Team systems get installed: breakout shape, neutral-zone reload, basic forecheck, defensive zone coverage. Walk-throughs first, controlled speed second, live last. One concept per week.

Phase 3 — Compete (Dec-Feb)

The longest phase. Refine systems, build compete habits, sharpen special teams. Game-transfer block dominates practice. Every drill ends in a battle, race, or scrimmage with a twist.

Phase 4 — Playoffs (Mar-Apr)

Short, sharp practices. Maintain conditioning, sharpen special teams, manage minor injuries. Add 30-60 seconds of off-ice film review per practice. Don't install anything new.

Weekly rhythm

What a typical 2-practice week looks like

Practice 1 (Tuesday)

Theme: Install or teach. Skill block dominates (40 minutes). Walk-through new concept slowly. Add controlled speed. Finish with a small-area game that uses the concept. Save compete finish for the end.

Practice 2 (Thursday)

Theme: Compete and live. Shorter skill block (20 minutes). Longer game-transfer block (30 minutes). All drills are live by the end. End with a 3v3 mini-tournament — winners get to skate first off.

Game day (Saturday)

Theme: Execute. Pre-game warm-up: edge work, passing star, quick shooting series. Post-game: 5-minute walk-around, then off-ice film while it's fresh. Save corrections for Tuesday.

Off day (Wed/Fri/Sun)

Theme: Recover. Optional off-ice work — stickhandling on a tile floor, light cardio, mobility. Players who play other sports use those days for that.

By age group

How the season template shifts from 8U to HS

8U

No "install" phase. Every practice is skill block + small-area game. Cross-ice mandatory. Phase model becomes: skating fundamentals → puck skills → compete games. Skip team systems entirely.

10U

Add a "support shape" theme in November. Introduce basic breakout (3 lanes). Avoid full PP/PK install — use "5v4 attack the slot" as a generic concept instead.

12U

Full phase model applies. Install full breakout in October, full forecheck in November. Special teams in January. Game-transfer block grows to 30 minutes by Phase 3.

14U / HS

Same phase model, faster install. Add film sessions weekly. Compete phase has 10 minutes of off-ice video per practice. Playoffs phase becomes "maintenance and matchups" — opponent-specific game plans.

For hockey directors

How an association installs season templates across teams

Standardizing across an association is harder than building one team's plan. The director's job isn't to write every team's practice — it's to give every coach the same scaffolding so quality stops varying by who happens to be coaching that team.

  • Provision each team with the season-template starter pack — phase model, monthly themes, weekly rhythm, sample drill swaps. The team coach picks the specific drills.
  • Run a 2-hour preseason coach clinic to walk through the template. The clinic is the install moment.
  • Push a templated practice once a month — usually a "Coach's Choice" drill block — so every team in the association runs the same skill rep on the same week.
  • Use an association director dashboard to see which teams are actually using the template (last practice saved, practices per week, AI builder usage). Without visibility, the install dies in November.
  • Capture the season template in shared club drills so coaches inherit the library every year, even when staff changes.
